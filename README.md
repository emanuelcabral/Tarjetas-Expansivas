# 🧩 Tarjetas Expansivas

Tarjetas Expansivas es un componente de interfaz interactivo basado en tarjetas que se expanden al hacer clic para mostrar contenido adicional. Está pensado para crear experiencias visuales modernas, limpias y dinámicas en la web.

---

## 🚀 Demo

Al abrir el proyecto, verás un conjunto de tarjetas visuales.  
Al hacer clic en cualquiera de ellas:

- La tarjeta se expande
- El resto se contrae
- Se muestra contenido adicional
- Puedes cerrarla con el botón de cierre

## ⚙️ Funcionamiento

Cada tarjeta:

- Se muestra en estado compacto por defecto
- Al hacer clic, recibe la clase `activa`
- Expande su tamaño y muestra contenido oculto
- Solo una tarjeta puede estar activa a la vez

---

## 🧠 Lógica principal

- Se seleccionan todas las tarjetas con `querySelectorAll`
- Se añade evento `click` para activar una tarjeta
- Se eliminan estados activos de las demás
- El botón de cerrar desactiva la tarjeta actual
